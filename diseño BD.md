# Diseno de base de datos - resumen

La documentacion completa del modelo se encuentra en:

[docs/diseno-base-datos.md](docs/diseno-base-datos.md)

El modelo se organiza en tres bloques principales:

1. Gestion de usuarios y acceso.
2. Recorridos, puntos de interes y contenidos.
3. Gamificacion, recursos digitales y CMS.

Esta estructura se corresponde con los requisitos funcionales y los casos de uso aprobados para el prototipo Iliturgi AR.
